Troubleshooting Guide for Keyence LJV7020

​This guide provides comprehensive solutions for diagnosing and resolving common issues related to hardware, desktop applications, connectivity, network configurations, and driver installations associated with Keyence LJV7020. By following the troubleshooting steps outlined, users can effectively address and rectify challenges encountered during the setup and operation of these 3D Lasers.

Hardware-Trouble Shooting Guide

Troubleshooting guide for power and connectivity Issues:

Indicator Status Possible Cause Trouble Shooting Steps
LINC/ACT LED OFF No network connection or
No power supplied. 		Check power supply and ensure the device is powered on.
Verify the network cable is properly connected.
Blinks Green Normal operation; data is being sent and received. No action needed.
Lights Green Device is linked to the network but not actively transmitting data. No action needed.
MS LED OFF No power to the EtherNet/IP unit. Check power supply and connections.
Lights Green Normal operation. No action needed.
Blinks Red Duplicated IP address.
Controller error. 		Check the IP settings and resolve conflicts.
Check controller status and error logs.
Lights Red System error. Restart the device.
Check logs for error details.
Contact support if the issue persists.
NS LED OFF No IP address set.
No power supplied. 		Ensure the device is powered on.
Assign a valid IP address.
Blinks Green IP address assigned, but no active connection.
 Verify network settings and ensure proper configuration.
Blinks Red Monitor data/External input" connection has not timed out. Check network stability.
Restart the device and re-establish the connection.
Lights Red Assigned IP address is already in use. Change the device’s IP address to avoid conflicts.

Troubleshooting guide for power and connectivity Issues:

Issue Possible Cause Solution
The POWER ON LED and ERROR LED do not light after switching ON the power of the controller. 1Is the power cable connected properly?
2.Power supply that not meeting the specification range?		 1.Connect the power cable properly..
2.Be sure to use a power supply that meets the specification range.
The power to the controller drops suddenly. Instantaneous power interruptions occurring? Use a stable power supply with no instantaneous power interruptions.
The power to the controller drops suddenly. Capacity of the power supply not meeting the specification range? If the capacity of the power supply is insufficient, replace it with a power supply with a greater capacity.
Nothing appears on the monitor Monitor switched OFF? Make sure that the monitor is on and use a power supply that meets the specification range of the monitor.
Nothing appears on the monitor. Monitor cable not connected properly? Connect the monitor cable properly.
Nothing appears on the monitor. Monitor cable broken? Try using another monitor cable.
Nothing appears on the monitor. Monitor adjusted properly? Adjust the brightness and hue of the monitor
Nothing appears on the monitor. Are the display unit and the controller connected properly using a cable? Switch the power OFF, check the connection between the display unit and the controller, and then switch the power back ON.
An error is displayed on the screen. Take the necessary measures by following the instructions in the displayed error message
The POWER ON LED of the controller does not light, but the ERROR LED is lit. Error message appear when using LJ-Navigator2 for communication? Error message appear on the monitor screen? Take the necessary measures by following the instructions in the displayed error message.
The laser emitting LED of the head does not light. Error message appear when using LJ-Navigator2 for communication? Error message appear on the monitor screen? Take the necessary measures by following the instructions in the displayed error message.
Laser beam is not emitted. Error message appear when using LJ-Navigator2 for communication? Error message appear on the monitor screen? Take the necessary measures by following the instructions in the displayed error message.
Laser beam is not emitted. Are the laser remote interlock and the laser OFF inputs being input correctly? Ensure that they are input correctly.
Laser beam is not emitted. Is the emission being controlled correctly according to the trigger mode setting? If the trigger mode setting is other than “Continuous trigger”, an external trigger input will be necessary.
The console cannot be operated. Is the console connected properly? Connect the console properly
The console cannot be operated. Is the console operation in locked state? Release the console operation lock.
The console cannot be operated. Is a PC or a PLC communicating with the controller? If multiple commands are input to the controller, only either of the commands will be accepted. (Console operation may be ignored.) Try operating the console again after making sure that no commands are being input from other instruments.
The PC freezes while using LJNavigator2. Does the system environment of the PC meet the specification range? Be sure to use it under a system environment that meets the specification range.
The profile is not displayed correctly. Is the target object located within the measurement range? Place the target object in the measurement range.
The profile is not displayed correctly. Is the laser beam being emitted? Place the target object in the measurement range.
The profile is not displayed correctly. Is the laser beam being emitted? Take the necessary measures by following the instructions in the displayed error message.
The profile is not displayed correctly. Is the laser beam being emitted? If the trigger mode setting is other than “Continuous trigger”, an external trigger input will be necessary.
The profile is not displayed correctly. If the averaging and median (time axis) are set to other than OFF, the profile will not appear until the specified number of triggers has been input. Enter a greater number of triggers than set for the averaging and median (time axis) counts.
The profile is not displayed correctly. Is the measurement being affected by a dead zone? Install the target object to a different location.
The profile is not displayed correctly. Is the measurement being affected by stray light caused by the ambient environment? 1.Install the target object to a different location or different angle.
2.Utilize the W polarization function.
3.Adjust the image setting correctly.
4.Utilize the image mask function.
The profile is not displayed correctly. Is the image setting adjusted properly? Adjust the image setting correctly.
The profile is not displayed correctly. Are the smoothing, averaging and median set correctly? Set the smoothing, averaging and median correctly.
The profile is not displayed correctly. Is the image mask area set correctly? Set the image mask area correctly.
The profile is not displayed correctly. Is there any dust, dirt, water or oil on the cover glass? Remove the dust, dirt, water or oil.
The profile is not displayed correctly. Are there water and/or oil droplets in the operating ambient environment? Remove the droplets using an air purge, etc.
The profile is not displayed correctly. Are there any scratches or cracks on the cover glass? The glass cover must be replaced. Contact your nearest KEYENCE office for repair.
The measurement value display does not change from decision standby data (“------”). Is trigger control being performed correctly according to the trigger mode setting? Perform trigger control according to the trigger mode setting.
The measurement value display does not change from decision standby data (“------”). Is timing control being performed correctly according to the hold mode setting? Perform timing control according to the hold mode setting.
The measurement value display does not change from decision standby data (“------”). Are the averaging count and the average number of times set correctly? Set the averaging count and the average number of times to appropriate values.
The measurement value is displayed as “ALARM”. Is the measurement mode set correctly? Set the measurement mode correctly.
The measurement value is displayed as “ALARM”. Is the position correction set correctly? Set the position correction correctly. If the position correction fails, the measurement values of all related OUTs will be ALARM values.
The measurement value is not output correctly. Is the position correction set correctly? Set the position correction correctly. If the position correction fails, the measurement values of all related OUTs will be ALARM values.
The measurement value is not output correctly. IAre the scaling, offset and ZERO standard value set correctly? Set the scaling, offset and ZERO standard value correctly.
The measurement value is not output correctly. Is the target object tilted or displaced? 1.Place the target object in the measurement range.
2.Set the tilt and height corrections.
3.Set the position correction.
Auto zero does not function correctly. Is the ZERO terminal allocation set correctly? Allocate the OUT correctly to the used ZERO terminal.
The measurement value is not output correctly. Is the auto zero key lock ON? Set the auto zero key lock to OFF.
Program No. cannot be changed. Is the program switching set correctly in the environment setting? Set it correctly according to the switching method.
Program No. cannot be changed. Is the key lock ON? Switch the key lock to OFF.
The judgement output is not output correctly. Is the wiring correct? On the terminal operation screen, check whether the ON/OFF state is correct. If the ON/OFF state is correct, check the output circuit and the wiring and connect them correctly.
The judgement output is not output correctly. Are the tolerance and judgement output terminal settings set correctly? On the terminal operation screen, check whether the ON/OFF state is correct. If the ON/OFF state is incorrect, set the tolerance and judgement output terminal settings correctly..
The judgement output is not output correctly. Is the import timing correct? Import it in the correct timing using the strobe output.
The analog output is not output correctly. Has the voltage exceeded +10.8 V or fallen below -10.8 V? Set the analog scaling correctly according to the measurement value.
