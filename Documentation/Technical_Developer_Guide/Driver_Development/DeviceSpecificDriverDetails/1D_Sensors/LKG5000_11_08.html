Introduction

Brief overview of the project

This project focuses on developing drivers for LK-G5000 to be utilized in automation applications, with a particular emphasis on integrating them with a user interface (UI) application for future use. The project aims to create software components that enable seamless communication between LK-G5000 and the UI application, allowing fast, accurate, and reliable measurements for various surfaces and materials, making it suitable for applications such as quality control, dimensional measurement, object detection, and In-line measurements. Through this integration, the project seeks to enhance the usability and functionality of LK-G5000 within automated processes, contributing to increased efficiency and productivity in various industrial settings.

Purpose of the document

The purpose of this document is to provide a detailed guide for developing a driver for the LK-G5000 device. The driver will enable communication between the Sensor and external software applications, allowing to get Judgement data, load and save programs, start and stop storage, and many more. It covers the software development process, including setting up the development environment, implementing the driver, testing and debugging, and integrating the driver with the LK-G5000.

The primary objective of the document is to provide a comprehensive guide for the developed drivers LK-G5000 used in automation using a UI application. This includes detailed instructions, a software installation guide, a device description with its features, and best practices to ensure the successful development and integration of drivers with the devices.

Device Overview

The Keyence LK-G5000 Series is an advanced line of laser displacement sensors designed for high-speed, high-precision, non-contact measurement applications across various industries

Description of the device for which the driver is being developed and keyfeatures

  1. Ultra High-Speed Sampling: Achieves a maximum sampling rate of 392 kHz, enabling the detection of minute changes in fast-moving targets. Keyence+1Laser Focus World+1
  2. Exceptional Accuracy:Linearity: ±0.02% of Full Scale (F.S.)
  3. Repeatability: As low as 0.005 µm, ensuring consistent measurements.
  4. Advanced RS-CMOS Sensor: Utilizes a high-speed CMOS sensor combined with an HDE lens pack to maintain focus across various materials and surfaces.
  5. Adaptive Light Control: Automatically adjusts laser intensity in real-time to accommodate different target surfaces, ensuring stable measurements.
  6. Versatile Sensor Head Options: Offers a range of sensor heads to suit different measurement ranges, accuracies, and beam spot sizes, catering to diverse application needs

System Components

  1. Sensor Heads: Various models available, including wide-type heads like the LK-H055, designed for specific measurement requirements.
  2. Controllers: Models such as the LK-G5001V feature built-in NPN output configurations, facilitating integration into different control systems.
  3. Configuration Software: LK-Navigator 2 provides an intuitive interface for sensor setup, calibration, and data analysis.
  4. Accessories: Includes monitors, interface units, extension units, power supply units, and optional parts to enhance system capabilities.

Typical Applications

The LK-G5000 Series is suitable for a wide range of applications, including

  1. Electronics: Measuring electrode thickness, connector heights, and zoom lens assembly accuracy.
  2. Automotive: Monitoring vibrations in disc rotors and ensuring precision in component assembly.
  3. Semiconductor: Assessing wafer surface heights and air gaps in touch panels.
  4. Manufacturing: Controlling positions of air knives and inspecting surface warpage.

Technical Specifications of LKG5000

  1. Display: TFT color LCD with 640 × 480 resolution, displaying 32,768 colors.
  2. Environmental Resistance: Front panel is IP65-rated, offering dust-proof and waterjet-proof protection.
  3. Operating Conditions:Temperature: 0 to 50°C (32 to 122°F)
  4. Operating Conditions:Humidity: 35 to 85% RH (non-condensing)
  5. Power Supply: DC 24V ±10%, with a current consumption of 1A or less.

Software Overview and Requirements – LK Navigator 2

LK-Navigator 2 is a configuration and monitoring software developed by Keyence for the LK-G5000 Series of ultra high-speed, high-accuracy laser displacement sensors. It facilitates seamless communication between a PC and the LK-G5000 controller, enabling users to configure parameters, monitor real-time data, and manage system settings efficiently.

Key Features

  1. Comprehensive Configuration: Users can upload, edit, and download settings between the PC and the controller. This includes configuring head settings, output (OUT) settings, common settings, and environmental parameters
  2. Real-Time Monitoring: The software allows for real-time display of measurement values and received light waveforms, aiding in immediate analysis and troubleshooting.
  3. Data Management: LK-Navigator 2 supports data storage functions, enabling users to start, stop, and clear data storage, configure storage methods, and upload/download settings.
  4. Program Handling: It provides functionalities to copy, initialize, and manage programs within the controller, streamlining the setup process for multiple controllers or configurations.
  5. User-Friendly Interface: The software features an intuitive interface with flow-of-operation guidance, making it accessible for users to navigate through various functions and settings.

System Requirements

To ensure optimal performance, the following system specifications are recommended:

  1. Operating System: Windows XP (SP3 or later), Windows Vista (SP2 or later), Windows 7 (SP1 or later), Windows 10, or Windows 11 Pro.
  2. CPU: Pentium III 1 GHz or higher (1.7 GHz or higher recommended)
  3. Memory: 256 MB or more (1 GB or more recommended).
  4. Hard Disk Space: At least 1 GB of available space.
  5. Display: 1024 x 768 pixels, 24-bit full color or better.

Connectivity Options

LK-Navigator 2 supports multiple connection methods to interface with the LK-G5000 controller: USB Connection: For direct and high-speed communication.

  1. Ethernet Connection: Allows for network-based communication, suitable for integrating into existing network infrastructures.
  2. RS-232C Connection: Provides serial communication, useful for legacy systems or specific integration requirements.

  5. GetStoredValues

    Description for GetStoredValues API: Retrieves the measurement data stored in the buffer of the LKG5000 and streams it over gRPC.

    RPC Command: rpc GetStorageData(GetStorageRequest) returns(GetStorageResponse) {}

    Request: message CommandRequest {
    string device_alias = 1;
    }
     message GetStorageRequest{
    .generic.CommandRequest cmd_request = 1;
    int32 out_no = 2;
    int32 time_storage = 3;
    LKIF_CALCMODE calc_mode=4;
    LKIF_TRIGGERMODE trigger_mode=5;
    string filepathInput = 6;}
    Parameter Description
    device_alias The unique device_alias used to refer the device
    out_no Channel Number from which data to be retrieved
    time_storage Time for which data needs to be retrived
    LKIF_CALCMODE calc_mode Calculation Mode for storage: Normal=0, Peak hold=1, Bottom hold=2,Peak-to-peak hold=3,Sample hold=4,Average hold=5
    LKIF_TRIGGERMODE trigger_mode Mode for triggering data storage:LKIF_TRIGGERMODE_EXT1=0,LKIF_TRIGGERMODE_EXT2=1.
    filepathInput File Name which is provided to store all data.
    Response:
    message CommandResponse {
    int32 resp_code = 1;
    string resp_description = 2;
    }
     message LKG5000DecisionList
    repeated LKG5000DecisionModel LKG5000_decision_models = 1;
    }
     message LKG5000DecisionModel {
    optional int32 OutNo = 1;
    optional LKIF_FLOATRESULT FloatResult = 2;
    optional float Value = 3;
    }
    Parameter Description
    FloatResult Measured Judgement Float Result, which provides result validation
    OutNo Out Channel Number from which data has been retrived
    Value Measurement value
    resp_code Response code from the device API
    resp_description Description of the response code
    CLI Command: python lkg5000.py GetStorageData lkg 1 1000 0 0 newfile
    Description for CLI Command:
    • GetStorageData: API call to the LKG5000 through CLI Command to Retrive Stored Measured Data from Provided channel for provided time
    • 1:Channel Number
    • 1000: Time in millisecond
    • 0: Measurement Mode(Normal=0, Peak hold=1, Bottom hold=2,Peak-to-peak hold=3,Sample hold=4,Average hold=5)
    • 0:Trigger Mode(LKIF_TRIGGERMODE_EXT1=0,LKIF_TRIGGERMODE_EXT2=1.)
    • newfile:User defined file name
    • lkg:Aliased Name of the device
