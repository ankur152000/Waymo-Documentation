Station Setup Flow Guide for User

  1. Overview

    The Waymo Automation Tool Station Setup is streamlined through a single batch file (.bat), ensuring a fully automated and repeatable environment setup process for all necessary development dependencies, services, and repositories.

    By double-clicking the batch file, the entire development environment is set up automatically — including the installation and configuration of essential components such as .NET, Python, Node.js, Git, and Node-RED. It also clones all required repositories, sets up virtual environments, install dependencies, and configures services — ensuring developers can start working immediately without spending time on manual setup or troubleshooting.

  2. PreRequisites for Station:

    Minimum Requirement for the Windows System :

    1. Windows 10 Pro
    2. 500 MB Free space on hard disk
    3. Ram 1 GB
    4. System Type 64-bit operating system, x64-based processor
    5. Processor:
      • Intel(R) Core(TM) i7-8700
      • CPU @ 3.20GHz 3.19 GHz
    6. Node JS Version: 23.11.0

  3. Prerequisites before executing .bat file

    Ensure the following files are in the same folder:

    1. dotnet-sdk-8.0.407-win-x64.exe
    2. Git-2.49.0-64-bit
    3. install-pre-requisites_v.3.0.1
    4. node-v23.11.0-x64
    5. python-installer
    6. node-v23.11.0-x64
    7. VC_redist.x64
    8. WaymoIcon

Automatic Installation Steps (Recommended)

  1. Step to install from WaymoArtifcats

    1. Download the WaymoArtifacts.zip folder and extract it. The folder contains all the necessary files for setup.

      2. Description text

    2. Run install-pre-requisites_v.3.0.1.bat as Administrator. This will automatically download and install all required dependencies.

      3. Description text


      Description text

    3. A Waymo icon will be created on your desktop and taskbar then double-click the Waymo icon to launch the application.

      4. Description text

    4. Now we have multiple options to start Node-RED:
      • For development run: go to Advanced → Start Node-RED
      • For production run: directly start Node-RED
      5. Option Description
      start gRPC Starts the gRPC Service, enabling communication with connected devices.
      stop gRPC Stops the gRPC Service, closing all active communication with connected devices.
      start node-red Starts the Node-RED.
      stop node-red Stops the Node-RED.
      exit Stops the gRPC service, closes all active connections with devices, and exits the system tray icon.
      logs Takes the user to the log file location. Three separate log files are maintained:
      • View gRPC logs
      • View Node-RED logs
      • View MFG-Inspector logs
      devices online Displays the number of devices currently online.

      Description text

    5. A login page will automatically open in your default browser.

      6. Description text

    6. Log in to begin using the Waymo Automation Tool.

Manual Installation Of Software

  1. Check and Install .NET SDK

    • Verifies if .NET SDK 8.0.407 is installed.
    • If not found, install using dotnet-sdk-8.0.407-win-x64.exe

  2. Check and Install Python

    1. Checks for Python 3.11.6.
    2. If not available, install Python in C:\Python3116.
    3. install pip, virtualenv, and creates a virtual environment.
    4. install necessary Python packages:
      • pip==24.2
      • grpcio-tools==1.60.0
      • pyModbusTCP==0.2.2
      • numpy==2.1.1

  3. Check and Install Node.js

    • Detects presence of Node.js.
    • If missing, install using node-v23.11.0-x64.msi.

  4. Install Microsoft Visual C++ Redistributable

    • install VC_redist.x64.exe

  5. Check and Install Git

    • Checks if Git is installed.
    • install Git-2.49.0-64-bit.exe if needed.

  6. Clone Required Repositories

    Clones the following repositories if not already present:

    • Waymo Node-RED flows: WaymoNodeRed
    • Waymo gRPC service: WaymoGrpc
    • Node-RED Project: Node-Red-Development

  7. Install and Verify Node-RED

    • install Node-RED globally using npm.
    • Verifies installation via node-red --version.

  8. Install Waymo Application

    • install WaymoIcon.msi (Waymo Service UI & System Tray).

  9. Install Node Modules

    Runs npm install in the following directories:

    • C:\NodeRed\.node-red\projects\Node-Red-Development
    • Moves node_modules to .node-red directory to fix dependency paths.
    • WaymoUI directories inside:
    • C:\Program Files (x86)\Creative Synergies Group\WaymoService\SystemIcon\WaymoUI
    • C:\WaymoApplication\WaymoGrpcService\WaymoGrpcService\SystemIcon\WaymoUI
  10. Important Note: If the setup .bat file is ever deleted or misplaced, you can find a backup copy in the following directory:
    C:\WaymoApplication\WaymoGrpcService\WaymoGrpcService\WaymoGrpcService\Batch Script
    Simply double-click the .bat file from this location to re-run the full station setup, including installations, repository cloning, virtual environment setup, and dependency installations.
